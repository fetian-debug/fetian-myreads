


# Fetian Myreads

This is web application [myreads](https://fetian-reads.web.app/) allow you to select and categorize books you have read, are currently reading, or want to read.

## Installation 

``` 
git clone https://github.com/fetian-debug/fetian-myreads.git

yarn 

yarn start

```
